Como avaliar o aprendizado?

Para avaliar seu aprendizado se auto avalie com estas quatro questões:
Obs: utilize o script feito anteriormente de calcular as médias, para cada questão, se auto avalie com uma nota de 0 até 10.

1)Seus código tem boa legibilidade, isto é, algum leigo na linguagem entenderia sem problemas?

2)Você seguiu os padrões de estilo da linguagem? Por exemplo, há um espaço antes e depois dos '=' de definição?

3)Você consegue imaginar outras maneiras de resolver essa questão? Seriam estas melhores?

4)Achou dificil completar os exercícios?

Caso as sua nota seja maior que 7.0, PARABÉNS, você conseguiu ter sucesso no aprendizado deste módulo de Python.

Caso contrário, busque mais exercícios, afinal, Python é uma linguagem em crescimento e seria bom profissionalmente saber manipular, pelo menos, alguma parte.
